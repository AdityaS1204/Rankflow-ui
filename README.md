# RankFlow UI

RankFlow UI is a premium, beautifully crafted React component library built with Tailwind CSS v4 and Framer Motion. It provides highly interactive, copy-and-paste UI blocks designed to accelerate the development of modern, aesthetically stunning applications.

📚 **[View Documentation & Components](/docs)**

## Contributing

We welcome contributions to grow our beautiful component registry! To contribute:

1. Setup: Run `npm install` and `npm run dev` in the `/app` directory.
2. Create your component in `app/registry/components/` using Tailwind CSS and `motion/react` (prioritize canvas/motion API for complex effects).
3. Register your component in both `app/registry/registry-ui.tsx` and `app/registry/index.ts`.
4. Run `npm run build-registry` to ensure your metadata compiles correctly.
5. Ensure your component elegantly supports both light and dark modes before submitting a Pull Request!
